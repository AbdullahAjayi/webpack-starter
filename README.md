### Webpack App Starter

---

Frontend enviroment for building JavaScript applications

Includes the following features:

- Babel Loader
- Sass Loader (Dart Sass)
- HTMLWebpackPlugin
- WebpackDevServer
- Assest Resource Loader
- Source Maps
- Caching

### Usage

Run Dev Server (Port 3000)

```
npm run dev
```

### Build for production

```
npm run build
```
